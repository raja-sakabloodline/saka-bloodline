# SAKA BLOODLINE

**A Malaysian psychological horror game inspired by Malay folklore, ancestral curses, and traditional supernatural beliefs.**

## About the Game

SAKA BLOODLINE takes players into a dark world shaped by inherited secrets, disturbing rituals, and a supernatural bloodline that refuses to remain buried.

Explore an atmospheric Malaysian setting, uncover the truth behind the family curse, and survive the presence that has followed the bloodline for generations.

## Game Information

- **Genre:** Psychological Horror
- **Platform:** Windows PC
- **Developer:** WorkPlayLabs
- **Languages:** Bahasa Melayu and English
- **Status:** In Development
